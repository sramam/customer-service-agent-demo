# F5 Distributed Cloud (XC) — Customer Overview

**Audience:** customers using SaaS-delivered WAAP, load balancing, and multi-cloud connectivity

## What it provides

F5 Distributed Cloud (often referenced as **XC**) delivers security and application networking as a service, including:

- **Web Application Firewall** with managed rule sets and bot defenses.
- **Global load balancing** across regions and cloud providers.
- **Mesh connectivity** between sites and cloud workloads with centralized policy.

## Common workflows

- Onboard an application by defining domains, origins, and TLS certificates.
- Attach WAF policies and tune exclusions based on observed false positives.
- Use analytics views to review threats and latency.
